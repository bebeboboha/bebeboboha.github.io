Hexo 加入 Disqus 來留言吧

發表於 2017-06-16 |

前言

發了幾篇的文章，發覺好像少了什麼東西～
啊！原來是留言回覆的地方啊！！
在Hexo中，就可以使用Disqus。

Disqus

Disqus是一款針對部落格、網站留言做統合的服務，使用者可以以匿名、Disqus、Facebook、Twitter、Google進流網站留言，同時幫你統計訪客留言排行榜，另外在留言下方也可以最近被熱門討論的文章，除此之外也可以把文章分享出去，並且留言中支援階層式(巢狀式)留言。

簡單來說，Disqus就是一個整合型的留言板啊！

註冊會員

  • 先到 Disqus首頁 註冊會員。
    • 點 GET STARTED
    • Signup 註冊會員
  • 設定留言板
    • 點選 I want to install Disqus on my site
    • 填寫設定 這邊需要記著Website Name -> 是下方URL前面那串
    • 選擇Basic
    • 一路順順的設定到Setup Complete!

Hexo 設定 Disqus

  • 開啟主題文件的_config.yml
  • 找到Disqus
    • enable修改為true
    • shortname填上剛剛記著的website name (URL版
  • clean、generate、deploy
  • 完成 🎉 🎉 🎉

尋找 Disqus 的 Shortname

如果忘記自己設定的Shortname，也可以找到呦～

  • 點右上角 Setting 的 Admin
  • 點選 Settings 找到創建的
  • Shortname 就找到囉

不要評論的文章

如果某些頁面需要取消評論，在md文件的前面front-matter中增加comments: false就可以囉。

完成

而一切設定完成，就會在文章最後出現留言的地方了！
