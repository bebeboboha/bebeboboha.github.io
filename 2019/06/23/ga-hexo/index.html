建立使用量分析的網頁

發表於 2019-06-23 |

使用量分析

點閱率

  • 較簡單，很多東西上都會做到點閱率。例：inVision, campsite
  • 就只有點閱率

GA google analytics

  • 豐富的分析，顯示訪客的性別、年齡層、興趣、國家等
  • 停留了多久時間，跳離率(Bounce Rate)多少
  • 流量在各時間的變化
  • 知道你的網站流量從哪裡來
  • 分析互動

實作想法

目前還想不到可以直接產url 並分析網站的工具，所以還是要：

  1. 先有網站網址
  2. 使用 google analytics
  3. 埋ga

網站網址

想以最低成本，且最方便的方式有一個可供測試的網站：

  1. Wix
  2. wordpress
  3. 購買domain
  4. git page

實際實作

Wix

在Wix 就內建 google analytics，註冊完ga 後取得追蹤id，準備要在wix中埋入ga時才發現，只有WiX進階付費版本才有此功能。

Business Basic US$17 / 月

wordpress

wordpress 有內建的外掛可以直接使用ga 或是修改header footer 來埋ga，但一樣是要升級至商務版方案才能安裝外掛程式。

商務版 US$25 / 月

好吧 既然都不給免費埋ga 那我們自己改寫應該行了吧

購買domain

寫了一個單純可以放ga 在head 的html 接著就是放上網域了
除了付費domain 外還有免費可以供測試的 freenom，但萬事俱備後才發現… 不對啊怎麼放?
還要在短時間內研究DNS研究如何發布，不行這要花太多時間了!!!
